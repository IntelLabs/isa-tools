# Welcome to the ISA Syntax Highlighter!

The ISA Syntax Highlighter is a powerful tool designed to enhance your coding experience with ISA (Intel&reg; Instruction Set Architecture Specification Language). With its basic syntax highlighting capabilities, you can easily identify different elements of your ISA code and write clean and error-free specification.

## Features

- Basic ISA syntax highlighting to help with coding.
